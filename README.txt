Place your PNGs in the assets folder:
- assets/background.png  (large image, e.g. 1600x1200)
- assets/player.png      (small sprite, e.g. 32x32 or 40x40)
- assets/object.png      (icon for interactable objects)

Run:
python main.py

Controls:
- Arrow keys or WASD to move
- E to interact when overlapping an object
- ESC to quit
